金融投资学

湖南大学 2023 年春季课程. 所有课程资料均会发布于此. 本课程讨论机器学习方法在金融经验研究中的应用.

参考书

[1] An Introduction to Statistical Learning, by Gareth James, Daniela Witten, Trevor Hastie, Robert Tibshirani. 豆瓣页面.

  • 英文第二版 PDF, 书籍官网链接 (境外网站, 可能需要科学上网)
  • 第一版中文翻译 PDF, 课程视频 bilibili 链接
  • 统计计算: 每个章节的末尾, 均有对应的 R 语言实操环节 (R Lab). 如果你写 python, 可以参考这个 github 项目. 我们课上只讲 R, 但你可以用任何你喜欢和擅长的工具来完成课程作业.

[2] R for Statistical Learning, 在线书籍.

[3] Machine Learning in Business: An Introduction to the World of Data Science, by John Hull.
